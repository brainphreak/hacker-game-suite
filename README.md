# Hacker Game Suite

A collection of four interactive games designed to teach and test cybersecurity skills in a fun and engaging way.

## Games

### 1. Identity Theft (OSINT Investigation)

An open-source intelligence (OSINT) game where you investigate targets by piecing together information from various social media profiles and forum posts.

- **Objective:** Answer questions about a target's personal life and background.
- **Skills:** OSINT, data correlation, and critical thinking.

### 2. Phishing Expedition (Phishing Detection)

A fast-paced game where you must identify phishing emails from legitimate ones within a 60-second time limit.

- **Objective:** Correctly classify as many emails as possible.
- **Skills:** Phishing detection, email analysis, and attention to detail.

### 3. Packet Sniffer (Network Forensics)

A network forensics puzzle where you intercept and decode hidden messages from captured network packets.

- **Objective:** Decode 5 packets to complete the mission.
- **Skills:** Network analysis, data decoding (Hex, Base64, ROT13, etc.), and pattern recognition.

### 4. Firewall Breach (Command Line Typing)

A typing game where you must quickly and accurately type cybersecurity-related commands to breach a firewall.

- **Objective:** Type a series of commands before the time runs out.
- **Skills:** Typing speed and accuracy, familiarity with common cybersecurity tools and commands.

## How to Play

Simply open the `index.html` file in your web browser to access the main menu and select a game to play.
